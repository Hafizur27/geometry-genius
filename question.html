Q1. What is Dom and what is the purpose of DOM ?

The meaning of 'Dom' is document object model. Dom is a programming interface for html document. Dom is very powerful because it can make a useful and dynamic web site. DOM represents a lot of useful purpose. For using dom we can add and remove any elements of our html document, can add any necessary information when we need and also it can control any style without using CSS. The DOM represents the document as nodes and objects that way, programming languages can interact with the page.

Q3. What is event bubble ?

Event Bubbling is happens when an element receives an event, and that event bubbles up to its parent and ancestor elements in the DOM tree until it gets to the root element. That is called event bubble. We can stop this event bubbling, using the stopPropagation() method.

Q3. What is a call back function and why will we use it ?

A call back function is a function that is passed as an argument to another function so that it can be executed in that other function is called as a callback function.
We need callback functions because many JavaScript actions are asynchronous, which means they do not really stop the program from running until they are completed.Thats why we need to use call back function.
